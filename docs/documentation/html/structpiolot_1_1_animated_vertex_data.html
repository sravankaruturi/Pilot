long piolot::AnimatedVertexData::header = 11122110

This contains the details regarding the following variables.

It can be a 8 - length integer value, signifying that the number of locations for now, is limited to 8. We can add increase them later on if needed. Each digit signifies the Data Type we are passing to the GPU. 0 –> No Value. 1 –> Float ( 32 bit?? ). 2 –> Int ( 32 bit?? ).

9 –> Undefined All the values are vec4's. The digits signify what the datatype of each of these vector elements is.
